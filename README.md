# 잠수함 게임 🐠🤿
------------

### ✔️ 게임 진행 방법
- enter키로 시작.
- 사방에서 날라오는 공을 방향키로 이동하며 오랫동안 피하면 된다!
- 공에 맞는 즉시 게임 종료!
- 오른쪽 상단에 점수가 표시 된다.  ( 살아남은 시간이 점수! )
- 다시 게임을 하고 싶으면 space바를 누른다.
